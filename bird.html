Convolutional Neural Network: Bird Species Classifier

Summary

The goal is to produce a classification model which predicts the species given an image of a bird. The model is a convolutional neural network. It is possible to reach high accuracy in relatively short time by taking advantage of transfer learning.

The species are selected as ones that are common in Southern Zrizona as this project was supported by the Tucson Bird Count. The list started as a subset of the 200 Birds Species by removing ones that have not been detected in that region. Common species from the Bird Count dataset were then added.

Model Development

In Keras it is straightforward to fine tune the standard Inception V3 network using transfer learning. Load the standard Inception V3 backbone but replace the top layer with a combination of Pooling followed by densely connected layers to produce one prediction score per species.

The dataset contains 16435 images across 110 species. That set is augmented by randomly adjusting zoom level and brightness, allowing a slight rotation and a horizontal flip. Train with batch size of 32, reserving 20% of the images at each step for validation. Accuracy converges after only two epochs.

Results

A subset of 639 images is held out from training so the model can be tested realistically. Species predictions of those images are compared to the actual, known labels to compute the model's average precision, recall and F1 score

Precision Recall F1 Score
97.5% 97.6% 97.3%

Application

To support real-world testing, it is convenient to have a Flask web app because it lets a wider range of people upload their own images and compare results to their own identification. This process can also find new species that should be added to the model.
