Data Analysis Examples

 

Feature Selection

Reduce the dimensionality of the dataset to a smaller set of core features that contribute the most to the overall objective. This significantly improves the ability of linear models to estimate a target value.

This example uses a large set of properties of various superconducting materials that can be used to estimate their critical temperature. Reducing the set of properties allows a simple linear relationship to estimate the critical temperature with high accuracy.

Convolutional Neural Network
