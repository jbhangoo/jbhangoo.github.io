Case Studies

 

Feature Selection

Reduce the dimensionality of the dataset to a smaller set of core features that contribute the most to the overall objective. Aggresive feature selection greatly improves the ability of linear models to predict the target value.

This example uses a large set of properties of various superconducting materials that can be used to estimate their critical temperature. Reducing the set of properties allows a simple linear relationship to estimate the critical temperature with high accuracy.

Supervised Learning

Use labeled data to train a model that can categorize future inputs with high accuracy.

This example uses measurements from Scania truck engines to predict failures. This example compares the accuracy of the results from a variety of machine learning models. The purpose is to demonstrate how accuracy is measured and evaluated for events that occur rarely.
