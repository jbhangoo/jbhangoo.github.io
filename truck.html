Machine Learning: Truck Engine Failure Prediction

Summary

The goal is to produce a classification model which predicts engine failure using supervised learning. This problem focuses on the rare failure case, so we want to see precision and recall details instead of just reporting a single value.

Data Set

Two samples were provided by Scania, a training set and a separate test set. The positive class consists of component failures for a specific component of the air pressure system (APS). The negative class consists of trucks with failures for components not related to the APS.

The datasets are anonymized and sampled from heavy Scania trucks in everyday usage. They were used for the Industrial Challenge 2016 at The 15th International Symposium on Intelligent Data Analysis (IDA).
